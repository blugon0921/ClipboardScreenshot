# ClipboardScreenshot

## English
Adds a 'Copy Screenshot to Clipboard' key.<br>
This key copies a screenshot directly to your clipboard without saving it as a file.<br>
Simply press the key, then paste it anywhere with Ctrl+V.

## 한국어
'스크린샷을 클립보드에 복사' 키를 추가합니다.<br>
해당 키는 스크린샷을 복사하고 파일로 저장하지 않습니다.<br>
해당 키를 누른 후 원하는 곳에 Ctrl+V로 붙여넣어 보내기만 하면 됩니다.
